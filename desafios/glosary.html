Estrutura HTML

!DOCTYPE html

Define o documento como HTMl:5.

HTMl

Define o documento como HTML.

Head

Container para informar links(CSS, JS), para definir icone da página, título, etc..

Body

Tag para o corpo do documento, aqui será escrito a maior parte do código.

h1/h2/h3...

Tag para fazer título e subtitulo.

strong

Tag para deixar em negrito.

dl/dt/dd

Tag para criar um glossário como este.

p

Tag para definir um paragrafo.

div

Tag para "separar um bloco de texto", com o objetivo de manipulalo, com css e js.

span

Outra tag para poder manipular, com css e js, porem esta serve para dar espaçamento na mesma linha.

a

Para fazer uma parte do texto como link.

img

Essa tag serve para inserir uma imagem.

figure

Essa tag serve para inserir uma imagem com legenda e contem um espaçamento próprio para poder se diferençicar da tag "img".
